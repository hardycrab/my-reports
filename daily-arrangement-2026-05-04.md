## 🎯 今日学习重点和实操任务

### 上午（9:00-12:00）

**理论学习和练习**：
1. **意图表达核心原则**（30 分钟）
   - 明确任务目标（要什么）
   - 提供充分背景（为什么）
   - 设定清晰约束（边界在哪）
   - 指定输出格式（长什么样）

2. **Prompt 结构拆解练习**（60 分钟）
   - 分析 3 个优秀 Prompt 案例
   - 识别其中的：角色设定 + 任务描述 + 约束条件 + 输出格式
   - 记录关键模式和可复用框架

3. **实操：重写模糊需求**（60 分钟）
   - 找出过去 3 天自己发出的 3 条模糊指令
   - 用今日所学重新改写，使其更清晰、可执行
   - 对比改写前后的效果差异

### 下午（14:00-17:00）

**实战应用**：
1. **场景化 Prompt 设计**（90 分钟）
   - 设计 3 个不同场景的 Prompt 模板：
     - 代码审查场景
     - 文档写作场景
     - 数据分析场景
   - 每个模板包含：角色 + 任务 + 输入 + 输出 + 约束

2. **迭代优化练习**（60 分钟）
   - 选择一个复杂任务（如：生成一份市场分析报告）
   - 进行 3 轮 Prompt 迭代
   - 记录每轮改进点和效果提升

### 晚上（19:30-21:00）

**复盘与沉淀**：
1. **今日收获总结**（30 分钟）
   - 记录 3 个最重要的学习点
   - 记录 2 个可以立即应用的技巧
   - 记录 1 个需要进一步探索的问题

2. **更新个人 Prompt 库**（30 分钟）
