## 💡 使用建议

### 新手入门路线
1. **Prompt Engineering 基础**：
   - 先看菜鸟教程入门
   - 再看吴恩达 1.5 小时课程
   - 最后阅读提示工程指南（中文版）

2. **区块链/Polymarket 入门**：
   - 先了解区块链基础（币安学院）
   - 再学习预测市场概念
   - 最后学习 Polymarket 平台使用

3. **开发实战**：
   - 阅读 Polymarket 官方文档
   - 参考 GitHub API 示例
   - 尝试调用 API 获取数据

### 学习技巧
- **每日 1-2 个链接**：不要贪多，深度学习
- **做笔记**：记录关键概念和代码示例
- **实践优先**：学完立即动手尝试
- **社区讨论**：遇到问题可在相关社区提问
